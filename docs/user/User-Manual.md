## 系统概述

老人监护管理系统是一个专为老年人健康监护设计的综合性平台，通过智能设备实时监测用户的健康状况，提供及时的预警通知和健康数据分析。

### 主要功能

- **实时健康监测**: 监测心率、血压、体温等关键健康指标
- **智能预警**: 异常数据自动预警，及时通知家人和医护人员
- **设备管理**: 管理各种健康监测设备
- **历史数据分析**: 查看健康数据趋势和统计分析
- **多渠道通知**: 支持短信、邮件、推送等多种通知方式

### 支持的设备类型

- 智能手环/手表
- 血压计
- 血糖仪
- 体温计
- 心电图设备
- 跌倒检测器

3. **设置通知**: 配置预警通知方式和接收人

## 用户注册与登录

### 注册新账户

1. 访问系统登录页面
2. 点击"注册新账户"按钮
3. 填写注册信息：
   - 用户名（3-20个字符）
   - 密码（至少8位，包含字母和数字）
   - 确认密码
   - 邮箱地址
   - 手机号码
   - 真实姓名
   - 选择用户类型（老年人/家属/医护人员）

4. 阅读并同意用户协议和隐私政策
5. 点击"注册"完成注册

6. 注册成功后，系统会发送验证邮件或短信，请及时验证。

### 登录系统

1. 在登录页面输入用户名和密码
2. 可选择"记住我"以便下次自动登录
3. 点击"登录"按钮
4. 如忘记密码，可点击"忘记密码"进行密码重置

### 账户安全设置

1. **修改密码**: 
   - 登录后进入"个人设置"
   - 选择"安全设置"
   - 输入当前密码和新密码

2. **绑定手机/邮箱**:
   - 确保手机号和邮箱已验证
   - 用于找回密码和接收重要通知

3. **登录历史**:
   - 查看最近的登录记录
   - 发现异常登录及时联系客服

## 设备管理

### 添加新设备

1. 进入"设备管理"页面
2. 点击"添加设备"按钮
3. 选择设备类型（智能手环、血压计等）
4. 填写设备信息：
   - 设备名称（自定义，如"我的手环"）
   - 设备ID（通常在设备标签上）
   - 设备位置（卧室、客厅等）
   - 所属用户（如果是家属管理）

5. 确认添加，系统将自动连接设备

### 设备连接方式

#### WiFi 设备
1. 确保设备已连接到家庭WiFi
2. 在系统中输入设备ID
3. 系统自动搜索并连接

#### 蓝牙设备
1. 确保手机蓝牙已开启
2. 在设备管理页面选择"蓝牙设备"
3. 按照设备说明书进行配对

#### 4G 设备
1. 输入设备SIM卡号码
2. 系统通过网络连接设备

### 设备状态监控

设备列表显示以下信息：

- **设备名称**: 自定义的设备名称
- **设备类型**: 手环、血压计等
- **在线状态**: 绿色表示在线，灰色表示离线
- **电量状态**: 电池电量百分比
- **最后更新**: 设备最后数据上传时间
- **设备位置**: 设备所在位置

### 设备故障处理

#### 设备离线
1. 检查设备电量是否充足
2. 确认设备是否在信号覆盖范围内
3. 重启设备
4. 如仍无法连接，联系技术支持

#### 数据异常
1. 检查设备佩戴是否正确
2. 确认设备是否需要校准
3. 查看设备使用说明书
4. 联系客服进行设备检测

### 设备维护

- **定期充电**: 保持设备电量充足
- **清洁保养**: 定期清洁设备传感器
- **软件更新**: 及时更新设备固件
- **校准检查**: 定期进行设备校准

## 健康监测

### 实时健康数据

系统实时显示以下健康指标：

#### 心率监测
- **正常范围**: 60-100次/分钟
- **显示内容**: 当前心率、趋势图表
- **预警设置**: 可设置高低阈值

#### 血压监测
- **正常范围**: 
  - 收缩压: 90-140 mmHg
  - 舒张压: 60-90 mmHg
- **显示内容**: 收缩压、舒张压、脉搏
- **测量提醒**: 可设置定时测量提醒

#### 体温监测
- **正常范围**: 36.0-37.2°C
- **显示内容**: 当前体温、变化趋势
- **发热预警**: 超过37.5°C自动预警

#### 血糖监测
- **正常范围**: 
  - 空腹: 3.9-6.1 mmol/L
  - 餐后2小时: <7.8 mmol/L
- **记录时间**: 餐前、餐后标记
- **趋势分析**: 血糖变化趋势图

#### 运动监测
- **步数统计**: 每日步数目标
- **活动时长**: 有效运动时间
- **卡路里消耗**: 每日消耗统计
- **睡眠质量**: 睡眠时长和质量分析

### 数据查看方式

#### 仪表板总览
- 显示最重要的健康指标
- 快速了解当前健康状况
- 异常数据突出显示

#### 详细数据页面
- 查看单个指标的详细数据
- 时间范围选择
- 数据导出功能

#### 图表展示
- 折线图显示趋势
- 柱状图显示分布
- 热力图显示时间模式

### 健康评分

系统根据各项健康指标计算综合健康评分：

- **评分范围**: 0-100分
- **评分标准**:
  - 90-100分: 优秀
  - 80-89分: 良好
  - 70-79分: 一般
  - 60-69分: 需要关注
  - 60分以下: 需要及时就医

## 预警管理

### 预警类型

#### 健康预警
- **心率异常**: 心率过高或过低
- **血压异常**: 血压超出正常范围
- **体温异常**: 体温过高或过低
- **血糖异常**: 血糖超出安全范围

#### 设备预警
- **设备离线**: 设备长时间未上传数据
- **电量不足**: 设备电量低于设定阈值
- **设备故障**: 设备检测到硬件问题

#### 行为预警
- **跌倒检测**: 检测到用户跌倒
- **长时间静止**: 用户长时间无活动
- **异常活动**: 异常的活动模式

### 预警级别

- **低级**: 一般提醒，建议关注
- **中级**: 需要注意，可能需要采取措施
- **高级**: 紧急情况，需要立即处理
- **紧急**: 危急情况，需要立即求助

### 预警处理流程

1. **预警触发**: 系统检测到异常数据
2. **通知发送**: 向预设联系人发送通知
3. **确认处理**: 收到通知后确认处理
4. **状态更新**: 更新预警处理状态
5. **后续跟进**: 记录处理结果和后续措施

### 预警设置

#### 自定义预警规则
1. 进入"预警设置"页面
2. 选择要设置的预警类型
3. 设置触发条件：
   - 阈值设置（如心率>120）
   - 持续时间（如持续5分钟）
   - 触发频率（避免重复预警）

#### 通知联系人设置
1. 添加紧急联系人
2. 设置通知优先级
3. 选择通知方式：
   - 短信通知
   - 电话通知
   - 推送通知
   - 邮件通知

### 预警历史查看

- **预警列表**: 按时间排序的预警记录
- **预警详情**: 触发原因、处理状态
- **统计分析**: 预警频率和类型分析
- **导出报告**: 生成预警报告文档

## 常见问题

### 账户相关

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，输入注册邮箱或手机号，按照提示重置密码。

**Q: 如何修改个人信息？**
A: 登录后进入"个人设置"，可以修改姓名、电话、邮箱等信息。

**Q: 如何删除账户？**
A: 联系客服申请删除账户，删除后数据无法恢复。

### 设备相关

**Q: 设备无法连接怎么办？**
A: 
1. 检查设备电量是否充足
2. 确认设备在信号覆盖范围内
3. 重启设备和路由器
4. 如仍无法解决，联系技术支持

**Q: 设备数据不准怎么办？**
A:
1. 检查设备佩戴是否正确
2. 确认设备需要校准
3. 参考设备说明书
4. 联系客服进行检测

**Q: 设备丢失怎么办？**
A: 
1. 立即联系客服挂失设备
2. 停止该设备的数据接收
3. 购买新设备并重新绑定

### 数据相关

**Q: 数据安全吗？**
A: 系统采用银行级加密技术，确保数据安全。只有授权人员可以访问您的健康数据。

**Q: 数据会丢失吗？**
A: 系统有完善的数据备份机制，数据不会丢失。

**Q: 如何导出历史数据？**
A: 在"历史数据"页面选择时间范围，点击"导出"按钮选择导出格式。

### 预警相关

**Q: 为什么没有收到预警通知？**
A:
1. 检查通知设置是否正确
2. 确认联系方式是否有效
3. 检查手机是否开启通知权限
4. 联系客服检查系统状态

**Q: 如何设置预警阈值？**
A: 在"预警设置"页面可以自定义各项健康指标的预警阈值。

**Q: 误报预警怎么办？**
A: 可以在预警记录中标记为误报，系统会学习并优化预警算法。
